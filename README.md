# 🚗 Car Price Prediction using Machine Learning

## 📌 Project Overview
Car price prediction is an important problem in the automobile industry. The price of a car depends on several factors such as brand value, fuel type, mileage, year of manufacture, and transmission type.

This project uses Machine Learning to predict the selling price of a car based on different features.

---

## 🎯 Objectives
- Analyze the car dataset
- Perform data preprocessing and cleaning
- Train a Machine Learning model
- Predict car prices based on input features
- Evaluate the model performance

---

## 📊 Dataset
The dataset used in this project contains information about different cars and their attributes.

Features in the dataset include:

- Car Name
- Year
- Selling Price
- Present Price
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission Type
- Number of Owners

Dataset Source: Kaggle
